# Kasa

This project is an application for renting apartments between private individuals.

## How to launch the project? 

This project is built with **React**.

## Getting started

### Installation
1. **Clone the project**

2. **Install npm packages**
```
npm install
```
3. **Start front-end server:** 
```
npm start
```
